SOME OF MY PROJECTS

Raycast 2D Platform Engine

A 2D platforming engine built in Unity3D, using raycasting for collision detection and movement instead of the default physics system. Based on an Official Unity tutorial, I implemented performance optimizations and clean code improvements to enhance maintainability and responsiveness. Project here!

2.5D FPS Engine

An experimental 2.5D FPS prototype built in Unity3D, designed with a strong focus on clean code, abstraction, and polymorphism. Features a raycast-based movement system and a modular, pluggable AI built on the state machine pattern, leveraging ScriptableObjects and interfaces for flexibility. 

The AI navigates using Unity’s NavMesh system and supports multiple behaviors, including patrolling, random coordinate searching, and chasing targets upon detection. Project here!

About Me
